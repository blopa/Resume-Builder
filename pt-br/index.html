O que é Resume Builder?

O Resume Builder é um projeto de código aberto gratuito que permite a qualquer pessoa manter e construir facilmente qualquer tipo de currículo usando o Google Spreadsheets ou jsonresume.org.

Como?

Faça uma cópia dessa Google Spreadsheet ou use um arquivo JSON do jsonresume.org e faça o upload aqui.

Porque?

Você sabe que tem as habilidades para o emprego ao qual está se candidatando e para mostrar que deve ajustar seu currículo para corresponder a essa vaga.

Para este trabalho, não importa que você tenha sido um policial espacial por 5 anos, eles só se importam se você pode ser um cuidador profissional de cães.

Então você abre seu editor de texto e começa a apagar coisas que você não precisa para esta vaga. Legal.

Mas então você foi rejeitado e uma nova vaga apareceu, mas desta vez eles se importam se você fosse uma Polícia Espacial. E agora? Você salvou o novo currículo como "resume-final-2.pdf "? Você ainda tem a versão antiga? OMG que dor!

Resume Builder para o resgate!

Com o Resume Builder, você pode ter um arquivo Spreadsheet ou JSON com todas as suas habilidades e experiências, e simplesmente ativar / desativar as que deseja através da interface do usuário do Resume Builder e salvar seu novo currículo incrível como PDF.
